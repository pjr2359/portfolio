About Rowing Analytics

This dashboard visualizes rowing performance data using the Gold Medal Standard (GMS) as a benchmark.

The Gold Medal Standard is a time-based performance metric used in rowing to compare an athlete's performance to international gold medal times.

Key Features:
  • Performance Comparison: Compare rowers based on their percentage of GMS
  • Race Analysis: View detailed race results and metrics
  • Filtering: Filter data by specific rower or race
  • Visual Representation: Interactive charts showing performance trends

Higher percentage values indicate better performance relative to gold medal standards.

About Modern Greek Conjugation

Modern Greek verbs have different conjugation patterns based on tense, person, and number.

This tool helps you practice conjugating common Modern Greek verbs through an interactive multiple-choice quiz.

Select a verb and tense, then try to identify the correct conjugation for each person. The tool will track your score and provide immediate feedback.

Example Verbs:
  • μιλάω (to speak)
  • αγαπάω (to love)
  • διαβάζω (to read)
  • γράφω (to write)
  • τρώω (to eat)
